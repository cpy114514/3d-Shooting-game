# Shadow Shooter

一款第三人称影子射击游戏。你可以自己打，也可以在无尽模式里一波一波撑下去，解锁升级，硬扛越来越强的敌人。

试玩页：
[https://cpy111.itch.io/shadow-shooter](https://cpy111.itch.io/shadow-shooter)

## 平台

- Web
- Windows

## 操作

- `WASD`：移动
- `鼠标`：转视角
- `左键`：攻击
- `右键`：瞄准
- `Shift`：冲刺
- `Space`：跳跃
- `Esc`：暂停
- `F11`：全屏

## 说明

- 主菜单可以进设置、教程和选关
- 设置支持画质、分辨率、音量、镜头距离等选项
- 无尽模式会按波次刷新敌人，清完一波后可以三选一升级
- 关卡、UI 和大部分参数都可以在场景里直接编辑
